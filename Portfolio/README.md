### Lab 3 - Inconvinient Drawing Tool

Finally, I created an inconvinient drawing tool. You can change the background color with the "space" key, delete everything with the "delete" (or "blankspace") key to start over and adjust the stroke weight with the slidder. The inconvinience starts when you try to choose a color, since the butons do allow you to change it, but the catch is that you don't know what color you'll be using now. And, also, that the stroke is slightly off. 

![Lab 04.3 Inconvinient Drawing Tool](/Portfolio/assets/Session_04/Lab_3.png)

Think of it in the following way, you are drawing with a poencil, as long as it is touching your paper (dragging the mouse without stopping) you'll be able to draw the stroke where it appears (it is not dirrectly in the mouse position, it can be between 100 and 200 pixels off from its position), but once you stop and pick up your pencil from the paper and try to start again, the strokes position changes. You never know where it's going to appear. 

The following video showcases how it works.

[![Lab 04.3 Inconvinient Drawing Tool](https://img.youtube.com/vi/-hl72mqsu4U/0.jpg)](https://www.youtube.com/watch?v=-hl72mqsu4U)

## FINAL PROJECT

### Session 9

After lasts session brainstorm I had made up my mind and I wanted to to the music oscilator, so I started working on the principal part of the code, creating the instruments. 

The first step was understanding the [p5.Oscillator()](https://p5js.org/reference/p5.sound/p5.Oscillator/) function that p5 has to create a Theremin, and how could I merge that with the [bodypose](https://docs.ml5js.org/#/reference/bodypose) option of m15, so that when my hands moved up and down, the frecuency changed. It was not that dificult since the example shown uses mousePressed() to change the frecuency on the canvas, so we just had to change that configuration to the bodypose recognition, specifying points 9 and 10 (wrists) to make the change in frecuency.

Once that was done, the challenge was making the oscillator sound as an specific instrument, a piano or a guitar. I didn't want to upload a ton of sound files and having them change depending on the wrist position in the Y axis of the canvas. Ithink it would have make it very difficult to calibrate and also, it wouldn't be as seamless as the oscillator, so I wanted to try and see if I could modify the frecuency to sound similar to one of those instruments.

Here is were I used the help of Claude.ai to understand how to modify the function and create the right code.

![Claude Answer 1](/Portfolio/assets/Session_09/ClaudeAnswer_1.png)

Even though I asked him for diferent instruments, at the end, I just used the Piano and the Guitar since it was easier to have one for each hand, and used audio samples for the drums. At last, I decided that I would use Teachable Machine to create the different poses that would allow you to activate the drums, but first, I would leave the oscillator

This first working session was spent mostly fixing the oscillator and figuring out how it would work. Also, I mapped out the workflow of the project.
